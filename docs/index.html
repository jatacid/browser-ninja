Instructions

Get started in minutes

Pick Elements

Highlight any element in your devtools that you want to modify.

Generate Snippets

Chat about your elements and Ninja will give you snippets to run.

Save & Run Code

Execute snippets directly from the chat, and save snippets to auto-run the next time you visit.

Yes, it's that easy!

  1. Get the Chrome extension download (as a .zip) and extract to a folder.
  2. Visit chrome://extensions/ and toggle on the 'Developer Mode' (top right corner)
  3. Click 'Load unpacked' (top left corner) and select your extracted folder.
  4. Optional - Pin the chrome extension's icon.
  5. Press the ninja extension icon on a webpage you want to edit!
  6. Using the extension is easy - simply add your api key, attach context to your chat by capturing your elements, and explain what kind of snippet you'd like.
